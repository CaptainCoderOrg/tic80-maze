Part 2: Collecting Keys

In this challenge, you will add code that detects when the player has stepped on to a key tile, removes the key tile from the map, and tracks the number of keys the player has collected.

Requirements

  1. Add at least one key to your maze
  2. The number of keys the player currently is visible at all times
  3. When the player steps on the key:
    • The key is removed from the map
    • The player's key count increases
